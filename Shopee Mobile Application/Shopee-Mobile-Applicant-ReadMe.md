# 🛍️ Shopee Mobile Application: E-Commerce QA
**Domain:** E-Commerce | Mobile Application (iOS/Android) | Payment & Logistics

The Shopee Mobile Application is a high-traffic e-commerce platform that handles millions of concurrent transactions. As a QA Specialist, I focused on the end-to-end (E2E) testing of the buyer's journey, ensuring that critical conversion paths—from product discovery to successful payment—remained defect-free across various mobile environments.

---

## 🛠️ My Role & Responsibilities
My primary objective was to ensure a frictionless user experience by conducting rigorous functional and regression testing on the app's core modules. I focused on verifying that complex promotional logic and payment integrations functioned correctly under various user scenarios.

### 🔍 Key QA Focus Areas:
* **Checkout & Payment Pipeline:** Validated the end-to-end transaction flow, including cart management, address selection, and payment gateway integration.
* **Voucher & Promotion Logic:** Tested the application of discount codes, flash sale pricing, and Shopee Coins to ensure accurate price calculations.
* **Mobile UI/UX Responsiveness:** Conducted cross-device testing to ensure the interface remained consistent and usable across different screen sizes and OS versions.
* **Search & Filter Accuracy:** Verified the relevance and functionality of product search algorithms and category filters.

---

## 🧪 Testing Methodology
E-commerce testing requires a focus on both functionality and user retention. I utilized the following strategies:

1. **Regression Testing:** Executed comprehensive regression suites following app updates to ensure that new features did not break existing checkout or login flows.
2. **Boundary Value Analysis:** Tested input fields (such as voucher codes and quantity selectors) to ensure the system handled minimum and maximum limits correctly.
3. **Negative Testing:** Simulated "unhappy path" scenarios, such as expired vouchers, insufficient wallet balance, and network interruptions during payment.
4. **User Acceptance Testing (UAT) Alignment:** Ensured all features met the specific business requirements and usability standards expected of a top-tier retail app.

## 📂 Documentation Standards
All testing activities were meticulously logged to provide clear insights into the app's stability.
